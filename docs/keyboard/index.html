Keyboard shortcuts

Every hotkey in the hfo TUI, grouped by scope. Global shortcuts work on every tab. Tab-specific keys appear in the footer bar as you navigate. All hotkeys are case-sensitive; uppercase means Shift is held.

Global (any tab)

KeysAction
1–6Jump to a tab (Dashboard · Models · Install · Tune · Help · Settings)
Tab · Shift+TabCycle tabs forward / backward
,Open Settings
?Open Help
Ctrl+HOpen the author's homepage in the default browser
q · Ctrl+CQuit (restores scrollback)

Dashboard

KeysAction
Alt+↑ · Alt+↓Focus a panel (full-screen zoom) / cycle through panels
Esc · bReturn to the 4-panel grid
EnterOpen the selected link in the default browser
rForce refresh (re-sample GPU / RAM / Ollama)

Models

KeysAction
↑ · ↓Navigate rows
LLaunch integration with the selected model (opens the launch menu pre-filled)
GLaunch menu without preset (pick model + integration manually)
IReinstall an orphan — regenerates the Modelfile from the GGUF if the record is gone
BCreate a .zip backup of the selected model (level-9 + metadata sidecar)
dRemove the tag from Ollama (shallow — keeps files)
Alt+dRemove the tag AND delete its on-disk folder (deep delete)
rRefresh list

Install (quick-confirm)

KeysAction
EnterInstall now with recommended defaults (score-picked quant + HF-tuned Modelfile)
OOpen the file browser to change the target directory
CCustomize Modelfile parameters (temperature, top_p, top_k, repeat_penalty, min_p, ctx)
EscBack to the quant picker

Tune

KeysAction
↑ ↓Navigate rows
← →Cycle enum values (for OLLAMA_KV_CACHE_TYPE, OLLAMA_FLASH_ATTENTION)
EEdit freely (freeform input for numeric values)
DSet to Default (Ollama's out-of-the-box value)
SSet to Suggested (hfo's ~90% capacity recommendation)
RReset all rows to Suggested
XReset all rows to Default
AApply — persist via setx/launchctl/systemd + restart the Ollama daemon

Settings

KeysAction
↑ ↓Navigate rows
EnterOpen dropdown (themes or 20 languages — searchable, type to filter)
EscClose dropdown without changes

Searchable dropdown: the Settings picker is shared across themes, languages, and sort order. Start typing to filter; ↑↓ navigates matches; Enter selects. Perfect when 20 languages won't fit on one screen.
